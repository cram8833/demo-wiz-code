# Readme
Application code used for the Marc's Wiz Code demo

```bash
npm install
```

Then to test built version:
```bash
node app.js
```

View in browser:
```bash
http://localhost:3000
```

And to test built version in a container locally

```bash
docker build -t sagesse-dev .
docker run -p 3000:3000 sagesse-dev
```

View in browser:
```bash
http://localhost:3000
```
